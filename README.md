# 8-RGB-POD-Profiles
SimHub LED Profiles For 8 RGB POD's

# Installation Instructions

1. Click the 'latest' Profile release on the right of this page and then select the '.ledsprofile' file from the list and choose a download location on your PC.
4. In SimHub select the 'Devices' tab and open your 8 RGB POD device settings.
5. In the 'Telemetry Leds effects' section, select 'Import Profile'. Navigate to your PC download location and select the '.ledsprofile' file.
6. Your new LED profile should now be loaded and '8 RGB POD - Default' should now be displayed in the 'Telemetry Leds effects' section.
7. Selecting 'Edit Profile' will allow you to select between 'Single','Left Hand' or 'Right Hand' POD profiles and then switch the sub profile modes (only one should be switched on at a time) or edit the sections to your own liking.
